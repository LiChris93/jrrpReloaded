## 服务端支持

- [x] Spigot 1.12+

## 命令列表([]为可选,<>为必填)

- /jrrp - 获取今日人品值
    - help - 获取命令帮助
    - clear [name] - 清空指定玩家数据/所有数据(仅OP可用)
    - get \<name\> - 获取指定玩家的人品值(仅OP可用)

## 变量列表

- %jrrp_num% - 人品值
- %jrrp_num_colored% - 带有颜色的人品值
